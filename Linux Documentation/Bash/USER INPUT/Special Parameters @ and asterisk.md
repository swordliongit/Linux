### @
Returns all of the positional parameters

### \"@"
Returns all of the positional parameters in this format:
```sh
"param1" "param2" "param3"
```

### \*
Returns all of the positional parameters

### \"\*"
Returns all of the positional parameters in this format:
```sh
param1IFSparam2IFSparam3
```
IFS variable will be placed between them.
